# WallGuide

## 前言

使用了`N`多个`VPN`软件，包括但不限于影梭、蓝灯、`MonoCloud`、加速器、`GreenVPN`、`borderSS`等等；同样的，也自建了好多种翻墙服务器，包括`SS/SSR/V2ray/Brook/Trojan`

相对而言，[Trojan](https://trojan-gfw.github.io/trojan/)提供了最稳定和最流畅的体验。所以本次教程也是基于`Trojan`的完整部署流程

## 架构

* 基本内容
* `Trojan`搭建
    * 服务端搭建
    * 客户端搭建
    * `Android App`
* 浏览器插件
* 命令行部署
